Setting Variables with let
The let function sets variables and returns the last variable. The output of any statistical function can be set
to a variable.
Below is a simple example setting three variables a, b and correlation.
let(a=array(1,2,3),
b=array(10, 20, 30),
correlation=corr(a, b))
