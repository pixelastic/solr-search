Tuples
The tuple is the next data structure we’ll explore.
The tuple function returns a map of name/value pairs. A tuple is a very flexible data structure that can hold
values that are strings, numerics, arrays and lists of tuples.
A tuple can be used to return a complex result from a statistical expression.
