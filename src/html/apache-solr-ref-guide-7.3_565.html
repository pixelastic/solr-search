Subquery Result Fields
To appear in subquery document list, a field should be specified in subquery’s fl parameter e.g., foo.fl (it
is not necessary to specify in main one’s fl). Of course, you can use wildcard in this parameter. For example,
if field title should appear in categories subquery, it can be done via one of these ways.
fl=...id,categories:[subquery]&categories.fl=title&categories.q=...
fl=...id,categories:[subquery]&categories.fl=*&categories.q=...
Subquery Parameters Shift
If a subquery is declared as fl=*,foo:[subquery], subquery parameters are prefixed with the given name
