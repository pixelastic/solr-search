We can manipulate arrays with functions. For example, we can reverse an array with the rev function:
rev(array(1, 2, 3))
Returns the following response:
{
"result-set": {
"docs": [
{
"return-value": [
3,
2,
1
]
},
{
"EOF": true,
"RESPONSE_TIME": 0
}
]
}
}
