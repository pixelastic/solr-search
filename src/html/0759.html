Lists
Next we have the list data structure.
The list function is a data structure that wraps streaming expressions and emits all the tuples from the
wrapped expressions as a single concatenated stream.
Below is an example of a list of tuples:
list(tuple(id=1, data=array(1, 2, 3)),
